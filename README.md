# IMDb Movie Review Sentiment Analysis

## Project Overview

This project performs Sentiment Analysis on IMDb movie reviews using Natural Language Processing (NLP) and Machine Learning techniques. The objective is to classify movie reviews as Positive or Negative based on their textual content.

The project includes data cleaning, text preprocessing, exploratory data analysis (EDA), feature engineering using TF-IDF, and sentiment classification.

## Features

* Text Data Cleaning
* HTML Tag Removal
* Stopword Removal
* Lemmatization
* Exploratory Data Analysis (EDA)
* TF-IDF Vectorization
* Sentiment Classification
* Model Evaluation

## Dataset

The dataset contains IMDb movie reviews and sentiment labels.

### Columns

* Review: Movie review text
* Sentiment: Positive or Negative
